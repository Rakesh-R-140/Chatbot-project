# 🤖 Chatbot Project (React + Vite)

## 📌 Project Overview

This is a chatbot application built using **React (Vite)**.
The chatbot responds to user messages using **custom JavaScript objects** and stores chat history using **localStorage**, so messages persist even after refreshing the page.

---

## 🚀 Features

* 💬 Interactive chat interface
* 🧠 Custom chatbot responses (object-based logic)
* 💾 Chat history saved using **localStorage**
* 🔄 Messages persist after page refresh
* ⚡ Fast performance using Vite
* 🎨 Simple and clean UI

---

## 🛠️ Tech Stack

* Frontend: React.js
* Build Tool: Vite
* Language: JavaScript
* Storage: localStorage
* Runtime: Node.js

## 📂 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/chatbot-project.git
```

2. Navigate to project folder:

```bash
cd chatbot-project
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
